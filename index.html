GitHub 加速下载代理

将 GitHub 链接转换为多区域加速链接，解决 GitHub 访问慢、下载失败等问题。

加速拉取链接

cloudflare (v4) 主站加速，全球高速分发！国内优选和IPv6网络支持请使用优选服务器。
Cloudflare (v4) 推荐 优选加速服务器，仅支持IPv4 网络智能解析。
Cloudflare (v4/v6) 优选加速服务器，支持 IPv6/IPv4 网络智能解析。
Fastly (v4) Fastly CDN 节点加速。

任一子加速节点不可用后预计会在 15 分钟内自动重定向至主站加速。

正确输入示例

本加速工具支持转换以下任意格式的 GitHub 资源链接。点击示例文本可自动填入上方输入框并转换：

# 分支源码
https://github.com/sky22333/hubproxy/archive/refs/heads/main.zip

# raw文件
https://github.com/sky22333/hubproxy/blob/main/install.sh
https://raw.githubusercontent.com/sky22333/hubproxy/refs/heads/main/install.sh

# Release源码
https://github.com/sky22333/hubproxy/archive/refs/tags/v1.1.8.zip

# Release文件
https://github.com/sky22333/hubproxy/releases/download/v1.1.8/hubproxy-v1.1.8-linux-amd64.tar.gz

# gist
https://gist.githubusercontent.com/oopsunix/2dbf20f64984773da6740d1d1cf7c2d4/raw/github-blacklist

# api
https://api.github.com/repos/umami-software/umami

# git
git clone https://gh-proxy.com/https://github.com/sky22333/hubproxy.git

#nightly.link
打开网站查看示例https://nightly.link/

功能特点

  • 支持代理访问 GitHub 的 Releases、Raw、Archive 等资源
  • 自动缓存，提升访问速度
  • 支持大文件下载，无限制
  • 无需配置，直接使用
  • 支持 HTTPS 安全访问
  • 多区域加速节点选择
  • 一键复制加速链接

支持的 GitHub 资源

  • Releases 附件下载 (release assets)
  • Raw 文件内容访问
  • 仓库打包下载 (archive)
  • Git Clone 操作 (部分支持)
  • Gist 原始文件
  • 项目文件直接下载
  • Git api 接口支持

注意事项

  • 本服务仅供学习研究使用
  • 请勿滥用，否则可能会被限制访问
  • 如果遇到问题，请检查链接格式是否正确
  • 建议收藏本站，以便日后使用
  • 转换工具会自动更新，无需手动升级
  • 选择最适合您网络环境的区域节点
